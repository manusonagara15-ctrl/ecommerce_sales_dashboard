📊 Madhav Ecommerce Sales Dashboard

🔹 Project Description
Developed an interactive Power BI Ecommerce Sales Dashboard.
Visualized sales data to provide meaningful business insights.
Used interactive filters and charts for easy data exploration.
Enabled quick analysis of sales performance across different dimensions.

🔹 Key Features
📌 Sales, Profit, Quantity, and Average Order Value (AOV) KPIs
📌 State-wise Sales Analysis
📌 Monthly Profit Trend
📌 Product Category-wise Quantity Analysis
📌 Customer-wise Sales Analysis
📌 Payment Mode Distribution
📌 Sub-Category Profit Analysis
📌 Quarter-wise Interactive Filter
📌 Dynamic and User-Friendly Dashboard
🔹 Tools & Technologies

Power BI
Power Query
DAX (Data Analysis Expressions)
Data Modeling
Data Visualization

🔹 Dashboard Insights
Identifies top-performing states based on sales.
Shows monthly profit trends for business monitoring.
Compares product category performance.
Analyzes customer purchasing behavior.
Displays preferred payment methods.
Highlights the most profitable product sub-categories.

🔹 Project Objective
Transform raw ecommerce data into interactive visual reports.
Help businesses monitor KPIs effectively.
Support data-driven decision-making.
Improve understanding of sales and profit performance.

Data Cleaning

Interactive Dashboard Design
Business Intelligence
Data Visualization
